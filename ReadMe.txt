Programa para Controle de Vendas e Estoque

Vendas&EstoqueDL
Release 1.2
 
<Portuguese>
 
Este é programa para controle de vendas e estoque desenvolvido em java, todo código está disponível.
 
<English>
 
This is a free software in java language to control sales and stock.
This project use the open source libraries swingx and substance. A data base backup are included, restore the backup by PosgreSQL Administrator. 

SwingX page: swingx.dev.java.net
Substance page: substance.dev.java.net
PostgreSQL page: www.postgresql.org

CVS Server: vendasestoquedl.cvs.sourceforge.net
Web Server: vendasestoquedl.sourceforge.net

You can import the zip file by Eclipse IDE 3.2 or later.
The libraries are included.
